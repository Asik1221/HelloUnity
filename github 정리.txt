토큰 생성 이후 권한은 변경 가능 함
유출에 주의 하자

1. github 가입
2. 깃허브 저장소 생성
3. 깃허브 저장소 클론(복제)
4. 내 컴퓨터의 내 문서 아래에 생성된 프로젝트
   디렉토리에서 파일 생성 및 작업완료
5. 커밋할 파일들 선택해서 스테이지에 올리기(Add)
6. 커밋하기 (commit) 보통은 버전명과 작업한 내용
7. 푸시하기 (push)
8. github.com/내아이디/내프로젝트 링크를 통해서
   정상적으로 업로드 되었는지 확인

github : VersionControlSystem의 한 종류이다.
게임의 세이브 포인트와 유사 한데 언제든지
저장 시점으로 돌아가서 작업이 가능하다.

커밋(commit) : 게임의 세이브에 해당하는 행동 git에서는 커밋이라 한다.
다시 말해서 언제든지 커밋한 시점으로 되돌아 갈 수 있다.
커밋을 할려면 저장을 원하는 파일들을 묶어서 커밋 명령을 수행하면 된다.
